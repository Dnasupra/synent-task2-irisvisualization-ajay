# Iris Dataset Visualization Project

## Problem Statement
The objective of this project is to analyze and visualize the Iris dataset to understand patterns, relationships, and differences between different flower species.

## Dataset Details
Dataset: Iris Dataset

The dataset contains information about iris flowers including:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
- Species/Class

## Approach
The following steps were performed:

- Loaded the Iris dataset
- Explored dataset structure and information
- Created visualizations to understand data patterns
- Compared different flower species using charts
- Analyzed relationships between features

## Results
- Visualized relationships between different features
- Identified differences between iris species
- Gained insights from graphical analysis

## Files Included
- Iris_Visualization.ipynb - Visualization notebook
- Iris.csv - Original dataset
